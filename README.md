# Сайт поддержки молодых предпринимателей в Удмуртии

Веб-платформа для поддержки подростков-предпринимателей в Удмуртской Республике с интеграцией Telegram бота и ИИ-рекомендациями.

## Структура проекта

- **Backend**: Node.js + Express + SQLite
- **Frontend**: React
- **База данных**: SQLite (можно легко перейти на PostgreSQL)

## Функциональность

### Основные разделы сайта:
1. **Главная страница** - приветствие и обзор разделов
2. **Информация об Удмуртии** - характеристика региона и предпринимательства
3. **Как открыть бизнес в 14 лет** - 10 советов и пошаговая инструкция
4. **Опыт успешных бизнесменов** - истории и интервью
5. **Бизнес-уроки** - видеоуроки и образовательные материалы
6. **Меры поддержки** - гранты, субсидии, программы
7. **Маркет молодых** - онлайн-магазин товаров от молодых предпринимателей
8. **Контакты** - форма обратной связи
9. **Путеводитель** - подкасты, гранты, наставники, акселераторы, партнеры, рекомендации

### Интеграция с Telegram:
- Авторизация через Telegram Web App
- Получение персональных рекомендаций от ИИ-бота
- Связь с путеводителем через рекомендации

## Установка и запуск

### 1. Установка зависимостей

```bash
npm run install-all
```

### 2. Настройка окружения

Создайте файл `.env` в корне проекта:
```
PORT=5000
NODE_ENV=development
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
TELEGRAM_BOT_USERNAME=your_bot_username
```

### 3. Запуск сервера

```bash
npm run dev
```

Сервер запустится на `http://localhost:5000`

### 4. Запуск клиента (в отдельном терминале)

```bash
npm run client
```

Клиент запустится на `http://localhost:3000`

## Структура базы данных

База данных SQLite создается автоматически при первом запуске. Включает таблицы:
- `users` - пользователи (Telegram авторизация)
- `recommendations` - рекомендации от ИИ-бота
- `podcasts` - подкасты
- `grants` - гранты
- `mentors` - наставники
- `accelerators` - акселераторы
- `partners` - партнеры (реклама)
- `products` - товары маркета
- `articles` - статьи и истории успеха
- `lessons` - бизнес-уроки
- `contacts` - обращения через форму контактов

## API Endpoints

### Авторизация
- `POST /api/auth/telegram` - авторизация через Telegram
- `GET /api/auth/user/:telegramId` - получить пользователя

### Контент
- `GET /api/content/articles/:category?` - получить статьи
- `GET /api/content/article/:id` - получить статью
- `GET /api/content/lessons/:category?` - получить уроки

### Путеводитель
- `GET /api/guide/podcasts` - подкасты
- `GET /api/guide/grants` - гранты
- `GET /api/guide/mentors` - наставники
- `GET /api/guide/accelerators` - акселераторы
- `GET /api/guide/partners` - партнеры
- `GET /api/guide/recommendations/:userId` - рекомендации

### Маркет
- `GET /api/market/products` - все товары
- `GET /api/market/product/:id` - товар по ID
- `GET /api/market/products/category/:category` - товары по категории
- `POST /api/market/product` - создать товар

### Контакты
- `POST /api/contacts/submit` - отправить обращение

## Интеграция с Telegram ботом

Для полной интеграции с Telegram ботом необходимо:

1. Создать бота через @BotFather
2. Получить токен бота
3. Настроить Web App в боте
4. Реализовать ИИ-логику для генерации рекомендаций на основе описания стартапа пользователя

Рекомендации сохраняются в базе данных и отображаются в разделе "Путеводитель" → "Рекомендации".

## Разработка

- **Backend**: Express.js для API

- **База данных**: SQLite (легко мигрировать на PostgreSQL)
