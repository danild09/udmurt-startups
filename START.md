# Инструкция по запуску проекта

## 🚀 Быстрый старт

### Важно: нужно запустить ДВА сервера!

1. **Backend сервер (API)** - порт 5000
2. **Frontend сервер (React)** - порт 3000

---

## 📋 Шаги запуска

### Вариант 1: Два отдельных терминала

**Терминал 1 - Backend:**
```bash
npm run dev
```
Сервер API будет доступен на: `http://localhost:5000`

**Терминал 2 - Frontend:**
```bash
npm run client
```
Сайт откроется автоматически на: `http://localhost:3000`

---

### Вариант 2: Один терминал (Windows)

Создайте файл `start-all.bat` и запустите его (см. ниже)

---

## 🌐 Доступ к приложению

- **Сайт (React)**: http://localhost:3000
- **API (Backend)**: http://localhost:5000/api

**Важно:** Открывайте сайт на порту **3000**, а не 5000!

---

## 🔧 Устранение проблем

### Ошибка "Cannot GET" на порту 5000
Это нормально! Порт 5000 - это только API. Откройте сайт на порту 3000.

### React не запускается
1. Убедитесь, что установлены зависимости: `cd client && npm install`
2. Проверьте, что порт 3000 свободен
3. Попробуйте удалить `client/node_modules` и переустановить

### База данных не создается
Запустите `reset-db.bat` для удаления старой базы данных
